# Blinkit Data Visualization Project
Project Overview

This project analyzes the Blinkit Product Dataset using Python and Data Visualization techniques. The objective is to explore product data, understand category distribution, pricing patterns, discounts, ratings, and derive meaningful business insights through graphical representations.

📊 Features
Data Cleaning and Exploration

Descriptive Statistics

Product Price Distribution Analysis

Category-wise Product Analysis

Visualization using Matplotlib and Seaborn

Business Insights from Blinkit Product Data


🛠️ Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Google colab

📂 Dataset

The project uses the Blinkit Product Dataset containing information such as:
Product Name

Category

Price

Discount

Rating

Other product-related attributes
